## About
This is my professional portfolio showcasing my work as a Full-stack Software Engineer and Web Developer. 
I specialize in building modern web applications using React, Node.js, PostgreSQL, and Express.

## Features
- Dark/Light mode toggle with localStorage persistence
- Responsive design optimized for all devices
- Portfolio section featuring 6 live projects with GitHub links
- CV download functionality
- Contact form with spam protection
- Skills showcase with technology logos
- Smooth animations using AOS (Animate On Scroll)
- Accessibility-focused design with proper contrast and focus states
